Los tes Primalres de la Web

Tim Berners-Lee Es considerado el padre de la WEB porque desarollo elementos basicos para el funcionamiento de la Web

HTML

HTML siglas de HyperText Makup Language, hace referencia al lenguaje marcado para la elaboracion de paginas web. Es un estandar que sirve de referencia para la elaboracion de paginas web en sus diferentes versiones, define una estructura básica y un código (denominado código HTML) para la definicion de contenido de una página web, sobre todo en lo referente a su escritura e interpretación. Es el lenguaje con el que se defina ñas páginas web.

Fuente: HTML. Wikipedia

HTTP


Hypertext Transfer Protocol o HTTP (en español protocolo de transferencia de hipertexto) es el protocolo usado en cada transicion de la World Wide Web, HTTP fue desarrollar por el Woe Wide Web Consortium y la Internet Engineering Task Foce, colaboración que culminó en 1999 con la publicacion de una serie de RFC, el más importante de ellos es el RFC 2616 que especifica la versión 1.1.

Fuente: HTTP. Wikipedia

URL

Un localizador de recursos uniforeme o URL- en ingles de Uniform Resoure Locutor - es un identificador de recursos uniforme (URI) cuyos recursos referidos pueden cambiar, esto es la dirección puede apuntar a recursos variables en el tiempo. Estan formados por una secuencia de caracteres, de acuerdo a un formato modélico y estandar, que designa recursos en una red, como internet
